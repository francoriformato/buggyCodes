What is buggyCodes?

buggyCodes is a platform where the user can access a variety of lessons about the programming logic.
Each lesson will start with the presentation of a real situation, showing what could go wrong if the associated code fails to work properly.
Subsequently, the user will be prompted to select which portion of the code has the bug that breaks its functionality.
Some examples may be an infinite loop caused by a badly implemented while-loop, a type conflict or a bad use of pointers.

Main features:

- Simple to understand code snippets in pseudo-language;
- Real situation that can be solved through coding;
- Funny examples to learn from;
- A stats log to track your progression;
- User friendly interface.
